# Exodus 666
- Authors: **Tyler Nguyen, William Obertone, Pablo Silva**
- Description: Final project for CSP

## Description
We were inspired by Touhou and Cuphead to create a similar bullet-hell game for our final project. The music was composed online through Online Sequencer. (https://onlinesequencer.net/) All code and media is original. Feel free to use and reproduce under our CC-0 license. (if that's even applicable?) Shoutout to LC (unknown by name for confidentiality) for making our demonic-inspired alphabet.

## Notes
- All imports are located in main.py
- There are some bugs due to latency in pygame (particularly the hitboxes of objects when there is a multitude thereof)


## Game Controls
- W/UP ARROW - Move UP
- A/LEFT ARROW - Move LEFT
- S/DOWN ARROW - Move DOWN
- D/RIGHT ARROW - Move RIGHT
- O/LEFT CLICK - Shoot
- E - Dash
